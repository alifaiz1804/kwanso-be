##Project Name
Kwanso BE App

##Tech Stack

1. ExpressJS
2. Typescript
3. Swagger
4. Mongo

##Setup Locally

1. Clone the repository

2. Install the dependencies using
   `npm i`

3. setup enviornments variables

```
PORT=5000

MONGO_URI="mongodb://localhost:27017/kwanso-be"

JWT_SECRET="secret123"
JWT_EXPIRE="1d"
```

4. Run command for seed
   `npm run seed`

5. Run the project
   `Npm run dev`
